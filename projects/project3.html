WEB APPLICATION

Todo App

This was my first project using flask on a webpage in combination with a simple and small database.

A simple Todo application built with Flask and SQLAlchemy that helps users manage their tasks in an easy way.

This project features task creation, completion tracking, and basic CRUD operations, making it a perfect introduction to me to web development with Python.

Stack:

  • Python Flask - Backend framework
  • SQLalchemy - Database ORM
  • HTML - Structure
  • CSS - Styling
  • Additional technologies:
    • SQLite - Database
    • Flask-Login - User authentication
    • Bootstrap - Frontend framework

The development process started with setting up a Flask environment and designing the database schema for tasks.

Key challenges included:

  • Implementing user authentication
  • Managing database relationships
  • Creating an intuitive user interface

The project successfully demonstrated:

  • Practical understanding of MVC architecture
  • Database management with SQLAlchemy
  • User authentication implementation
  • Frontend and backend integration
