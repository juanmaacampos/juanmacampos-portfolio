Server Infrastructure

Home Server

Local server setup for networking and hosting

A home server infrastructure project focused on creating a centralized system for file sharing, media streaming, and web hosting capabilities within a local network environment.

The server provides essential network services including file sharing through Samba, web application hosting via Flask, and secure remote access through SSH, all protected by UFW firewall configurations.

Stack:

  • Ubuntu Server - Main operating system
  • Flask - Web application framework for hosting internal tools
  • Python - Backend scripting and automation
  • SSH - Secure remote access and management
  • Samba - Network file sharing service
  • UFW - Uncomplicated Firewall for security

Additional Tools:

  • NGINX - Reverse proxy server
  • Docker - Container management
  • Portainer - Container visualization and management

Project Implementation:

  • Set up Ubuntu Server with basic security configurations
  • Configured network services and port forwarding
  • Implemented Samba shares with user authentication
  • Developed custom Flask applications for internal tools
  • Created Docker containers for service isolation
  • Established backup and monitoring systems

Challenges Overcome:

  • Network security hardening
  • Service permission management
  • Resource optimization for 24/7 operation

  • Established a platform for hosting personal projects
